┌─────────────────────────────────────────────────────────────┐
│ STEP 1: Pre-Installation                                    │
└─────────────────────────────────────────────────────────────┘
   ├─ Read: CHECKLIST.md
   ├─ Ensure: Internet connection
   ├─ Verify: Sudo privileges
   └─ Check: 5GB+ free space

              ↓

┌─────────────────────────────────────────────────────────────┐
│ STEP 2: Run Installer                                       │
└─────────────────────────────────────────────────────────────┘
   
   $ ./install.sh
   
   This will:
   ┌───────────────────────────────────────────────────┐
   │ [1] Update system                                 │
   │ [2] Install 60+ packages from official repos     │
   │ [3] Install paru (AUR helper)                    │
   │ [4] Install AUR packages (ghostty, breeze)       │
   │ [5] Setup Flatpak + MissionCenter                │
   │ [6] Extract Kora icon theme                      │
   │ [7] Copy Hyprland configs                        │
   │ [8] Copy Waybar configs + scripts                │
   │ [9] Copy Rofi config                             │
   │ [10] Setup wallpapers (dynamic username)         │
   │ [11] Configure GTK dark theme                    │
   │ [12] Setup greetd login manager                  │
   │ [13] Add user to groups                          │
   └───────────────────────────────────────────────────┘

              ↓

┌─────────────────────────────────────────────────────────────┐
│ STEP 3: Installation Complete                               │
└─────────────────────────────────────────────────────────────┘
   
   ✅ All packages installed
   ✅ Configurations copied
   ✅ Theme configured
   ✅ Scripts ready
   
   Your setup is at:
   ├─ ~/.config/hypr/           (Hyprland)
   ├─ ~/.config/waybar/         (Waybar + scripts)
   ├─ ~/.config/rofi/           (Rofi)
   ├─ ~/.config/gtk-3.0/        (GTK theme)
   ├─ ~/.config/gtk-4.0/        (GTK theme)
   ├─ ~/.local/share/icons/     (Kora icons)
   └─ ~/Pictures/wallpapers/    (Wallpapers)

              ↓

┌─────────────────────────────────────────────────────────────┐
│ STEP 4: Reboot                                               │
└─────────────────────────────────────────────────────────────┘
   
   $ reboot

              ↓

┌─────────────────────────────────────────────────────────────┐
│ STEP 5: Login Screen (TuiGreet)                             │
└─────────────────────────────────────────────────────────────┘
   
   1. Enter your username
   2. Enter your password
   3. Select: Hyprland
   4. Press Enter

              ↓

┌─────────────────────────────────────────────────────────────┐
│ STEP 6: Welcome to Hyprland! 🚀                             │
└─────────────────────────────────────────────────────────────┘
   
   You should see:
   ┌────────────────────────────────────────────┐
   │  Waybar at top with:                       │
   │  [Workspaces] [Window] [Clock] [Modules]   │
   └────────────────────────────────────────────┘
   
   Your wallpaper loaded ✓
   Dark theme applied ✓
   Kora icons active ✓

              ↓

┌─────────────────────────────────────────────────────────────┐
│ STEP 7: Start Using                                         │
└─────────────────────────────────────────────────────────────┘
   
   Essential commands:
   
   SUPER + R  →  Launch Rofi (app launcher)
   SUPER + T  →  Open Kitty terminal
   SUPER + E  →  Open Nautilus (file manager)
   SUPER + Q  →  Close window
   SUPER + L  →  Lock screen
   
   📖 See QUICKSTART.md for more keybindings
